Installationsanleitung:

1. Sorgen Sie dafür, dass sich eine aktuelle JAVA-Version auf dem Rechner befindet. 
1. Entpacken Sie die DateiWatkost4.zip
2. Kopieren Sie den entpackten Ordner "Watkost4" bitte direkt auf C:\.
3. Durch Anklicken des Watkost-Icons öffnen Sie die ausführbare Watkost-Datei. Erst wenn Sie jetzt eine Eingabemaske
   anklicken, verlngt das Programm ein für 2014 gültiges Passwort. Zum Beginn des Kalenderjahres 2015 benötigen 
   Sie ein neus Passwort.

Die wesentlichen Änderungen gegenüber der Version 3.1.1 sind:
	Die Anlagenkosten sind marktüblichen Preisen 2014 angepasst.
	Anlagenfabrikatsbezeichnungen sind weitgehend entfernt.
	Das Berechnungsprogramm ist auch bei der Verwendung von salzreicherem Wasser (Brackwasserqualität) möglich.
	

Eine ausführliche Programmbeschreibung befindet sich als pdf-Datei im Ordner Watkost4.
